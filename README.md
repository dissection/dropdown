# dropdown

## options

**trigger** [`Boolean`]  
   false :默认为 盒子下面的 item 元素 ,true: _this.el自身  
 [`default`] !1
 
**item** [`string`]  
  下拉包含元素下， 的头部元素 ,也就是 移入 或者 点击 的元素  
[`default`] "ui-dropdown-item"
 
**current** [`string`]  
  当前 移入元素所添加的calss  
[`default`] "ui-dropdown-hover"

**enterDelay** [`number`]  
    移入触发动作的时间  
[`default`] 0

**leaveDelay** [`number`]   
    移出触发动作的时间    
[`default`] 0

**event** [`string`]  
    移入事件 和 点击事件
[`default`] mouseenter    

**onchange** [`function`]  
    改变时触发  
[`default`] null  
 
**onmouseleave** [`function`]  
    移出时触发  
[`default`] null
